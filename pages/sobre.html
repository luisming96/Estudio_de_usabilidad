Sobre VitaClick

VitaClick es una aplicación web orientada a facilitar la gestión diaria de la medicación y a mejorar la comunicación entre pacientes y profesionales sanitarios, poniendo el foco en la usabilidad, la accesibilidad y una experiencia de usuario clara e intuitiva.

HTML5 CSS3 JavaScript Bootstrap 5 UX/UI Accesibilidad Web

Proyecto académico desarrollado como parte del ciclo formativo de Grado Superior en Desarrollo de Aplicaciones Web.

Práctica de Diseño de Interfaces

Esta aplicación web forma parte de una práctica del módulo de Diseño de Interfaces, cuyo objetivo principal es realizar un estudio de usabilidad, accesibilidad y diseño responsive aplicado a un caso real.

Para ello, se han implementado tres skins visuales diferentes, permitiendo analizar cómo el diseño influye en la experiencia de usuario y en la accesibilidad de la interfaz.

Tecnologías utilizadas: HTML5, CSS3, JavaScript y Bootstrap 5.
